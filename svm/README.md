## Introduction

实验采用MNIST手写数字识别数据集，通过对图片进行HOG特征提取后使用SVM进行训练，得到的模型在测试集上的准确率达到了98.97%.

---
SVM参数设置：C=5，gamma=0.34， kernel='rbf'
